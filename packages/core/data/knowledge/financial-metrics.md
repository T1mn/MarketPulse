# 财务指标

本文档介绍投资分析中常用的财务和估值指标，帮助用户评估资产价值和投资机会。

---

## 市盈率 (P/E Ratio)

### 核心概念

市盈率（Price-to-Earnings Ratio）是最常用的股票估值指标，反映投资者愿意为每单位盈利支付多少价格。

### 计算公式

```
市盈率 = 股价 / 每股收益 (EPS)
       = 市值 / 净利润
```

### 类型区分

1. **静态市盈率 (Trailing P/E)**
   - 使用过去12个月的实际盈利
   - 数据确定，无预测风险
   - 可能不反映未来变化

2. **动态市盈率 (Forward P/E)**
   - 使用未来12个月的预期盈利
   - 反映市场预期
   - 依赖盈利预测的准确性

3. **滚动市盈率 (TTM P/E)**
   - 使用最近四个季度的盈利
   - 相对及时的估值参考
   - 最常用的市盈率类型

### 解读方法

**一般参考范围：**
- P/E < 15：可能被低估或增长缓慢
- P/E 15-25：合理估值区间
- P/E > 25：可能被高估或高成长

**注意事项：**
- 不同行业P/E水平差异大
- 高成长公司通常有较高P/E
- 周期性行业P/E波动大
- 亏损公司无法计算P/E

### 行业比较

| 行业 | 典型P/E范围 | 特点 |
|------|-------------|------|
| 科技 | 20-40 | 高成长预期 |
| 金融 | 8-15 | 稳定但低成长 |
| 消费 | 15-25 | 稳定现金流 |
| 公用事业 | 12-18 | 低成长，高股息 |

### 投资应用

1. **纵向比较**：与公司历史P/E比较
2. **横向比较**：与同行业公司比较
3. **市场比较**：与市场平均P/E比较

### 局限性

- 盈利可被会计手法调节
- 不适用于亏损公司
- 忽略了公司债务情况
- 一次性收益/损失会扭曲P/E

## 市净率 (P/B Ratio)

### 核心概念

市净率（Price-to-Book Ratio）衡量股价相对于每股净资产的倍数，反映市场对公司资产价值的评估。

### 计算公式

```
市净率 = 股价 / 每股净资产
       = 市值 / 净资产（股东权益）
```

### 解读方法

**参考范围：**
- P/B < 1：股价低于账面价值，可能被低估
- P/B = 1-3：合理区间，视行业而定
- P/B > 3：市场给予较高溢价

**P/B < 1 的情况：**
- 可能存在资产减值风险
- 可能是价值投资机会
- 需要分析具体原因

### 行业适用性

**适合使用P/B的行业：**
- 银行业：资产主要是金融资产
- 保险业：投资组合估值重要
- 房地产：土地和物业价值

**不太适合的行业：**
- 科技公司：核心资产是无形的
- 服务业：轻资产运营
- 高成长公司：未来价值未体现在账面

### 与ROE的关系

```
P/B = ROE × P/E
```

- 高ROE公司通常有较高P/B
- P/B反映市场对公司盈利能力的预期
- ROE持续下降会导致P/B回归

### 投资应用

1. **银行股估值**
   - 通常用P/B而非P/E
   - P/B < 0.8可能有投资价值
   - 需要考虑资产质量

2. **破净股筛选**
   - P/B < 1的股票
   - 可能是深度价值投资标的
   - 需要排除基本面恶化的公司

3. **周期股估值**
   - 周期底部盈利低，P/E失真
   - P/B相对稳定
   - 更适合估值参考

## 市值 (Market Cap)

### 核心概念

市值（Market Capitalization）是公司在市场上的总价值，等于股价乘以总股本，是衡量公司规模的核心指标。

### 计算公式

```
市值 = 当前股价 × 总股本
```

**加密货币的市值：**
```
市值 = 当前价格 × 流通供应量
```

### 市值分类

**传统股票市场：**

| 分类 | 市值范围（美元） | 特点 |
|------|------------------|------|
| 大盘股 | > 100亿 | 稳定，流动性好 |
| 中盘股 | 20-100亿 | 成长潜力 |
| 小盘股 | 3-20亿 | 波动大，机会多 |
| 微盘股 | < 3亿 | 风险高 |

**加密货币市场：**

| 分类 | 市值范围（美元） | 特点 |
|------|------------------|------|
| 大盘币 | > 100亿 | BTC、ETH等 |
| 中盘币 | 10-100亿 | 主流山寨币 |
| 小盘币 | 1-10亿 | 波动大 |
| 微盘币 | < 1亿 | 高风险高回报 |

### 市值的意义

1. **公司规模**
   - 反映公司当前市场估值
   - 决定股票的流动性
   - 影响指数权重

2. **投资决策**
   - 不同规模股票风险收益特征不同
   - 组合配置的参考依据
   - 衡量投资机会大小

3. **行业对比**
   - 同行业内比较相对地位
   - 判断市场集中度
   - 评估竞争格局

### 注意事项

- 市值会随股价波动
- 不等于公司的内在价值
- 需要考虑债务情况（企业价值）
- 加密货币市值可能有操纵

## 总锁仓量 TVL (Total Value Locked)

### 核心概念

TVL是DeFi领域的核心指标，表示锁定在某个协议或整个DeFi生态中的加密资产总价值，反映用户对协议的信任和使用程度。

### 计算方法

```
TVL = Σ(各类资产数量 × 各类资产价格)
```

包括：
- 流动性池中的资产
- 借贷协议中的抵押品
- 质押的代币
- 其他锁定资产

### TVL的意义

1. **协议健康度**
   - TVL越高，协议越受信任
   - TVL稳定增长是积极信号
   - TVL骤降可能有问题

2. **生态发展**
   - 衡量DeFi整体规模
   - 跨协议比较的基准
   - 链上活动的指标

3. **投资参考**
   - 筛选优质协议
   - 评估代币价值
   - 识别新兴机会

### 相关指标

1. **市值/TVL比率**
   ```
   市值/TVL = 协议代币市值 / TVL
   ```
   - 比率 < 1：可能被低估
   - 比率 > 1：市场给予溢价
   - 用于横向比较协议估值

2. **TVL增长率**
   - 日/周/月TVL变化
   - 反映资金流入流出
   - 趋势比绝对值重要

3. **每用户TVL**
   ```
   每用户TVL = TVL / 活跃用户数
   ```
   - 衡量用户质量
   - 鲸鱼占比高低

### 查询工具

- DefiLlama：最全面的TVL追踪
- DeFi Pulse：以太坊DeFi数据
- 各链官方仪表盘

### 注意事项

- 同一资产可能在多协议中重复计算
- 代币价格波动影响TVL
- 新协议可能通过高激励吸引TVL
- TVL高不等于协议安全或收益高

## 完全稀释估值 FDV (Fully Diluted Valuation)

### 核心概念

FDV是假设所有代币都进入流通后的总市值，包括尚未释放的代币，反映项目的潜在最大估值。

### 计算公式

```
FDV = 当前价格 × 最大供应量
市值 = 当前价格 × 流通供应量
```

### 与市值的关系

```
FDV/市值 比率 = 最大供应量 / 流通供应量
```

**比率解读：**
- 比率接近1：大部分代币已流通
- 比率很高：大量代币待释放，有稀释压力

### 为什么FDV重要

1. **估值参考**
   - 反映项目"真实"估值
   - 与同类项目横向比较
   - 避免被低流通量误导

2. **稀释风险评估**
   - 计算未来卖压
   - 评估代币经济模型
   - 判断当前价格合理性

3. **长期投资决策**
   - FDV/收入比率
   - FDV/TVL比率
   - 长期价值评估

### 实际案例

**案例1：高FDV/市值比率**
- 代币价格：$10
- 流通量：1000万（市值1亿）
- 最大供应：1亿（FDV 10亿）
- 比率：10倍
- 风险：大量代币释放可能压制价格

**案例2：低FDV/市值比率**
- 代币价格：$10
- 流通量：9000万（市值9亿）
- 最大供应：1亿（FDV 10亿）
- 比率：1.1倍
- 相对安全：稀释压力小

### 投资建议

- FDV/市值比率超过5倍需谨慎
- 了解代币释放时间表
- 关注团队/投资者代币锁仓期
- 评估未来卖压的来源和时间

## 流通量与总量

### 核心概念

**流通供应量（Circulating Supply）**：当前在市场上可自由交易的代币数量

**总供应量（Total Supply）**：已创建但可能包含锁仓的代币总数

**最大供应量（Max Supply）**：代币可能存在的最大数量上限

### 关系说明

```
流通供应量 ≤ 总供应量 ≤ 最大供应量
```

**区别说明：**
- 流通量：可以在市场交易
- 总量 - 流通量：锁仓、质押、销毁地址中
- 最大量 - 总量：尚未创建（如待挖矿的BTC）

### 不同模型

1. **固定最大供应**
   - 比特币：2100万枚
   - 提供稀缺性保证
   - 通缩属性

2. **无限供应**
   - 以太坊（合并前）
   - 通过其他机制控制通胀
   - 需要关注发行率

3. **通缩机制**
   - 代币销毁（Burn）
   - 回购销毁
   - 以太坊EIP-1559

### 对价格的影响

1. **供应减少**
   - 销毁、锁仓增加
   - 在需求不变情况下利好价格
   - 通缩叙事

2. **供应增加**
   - 解锁、增发
   - 产生卖压
   - 需要新需求消化

### 关键指标

1. **流通率**
   ```
   流通率 = 流通供应量 / 最大供应量
   ```
   - 高流通率：稀释压力小
   - 低流通率：潜在卖压大

2. **年通胀率**
   ```
   通胀率 = 年新增供应 / 当前总供应
   ```
   - 衡量稀释速度
   - 与质押收益比较

3. **解锁时间表**
   - 团队代币何时解锁
   - 投资者代币释放
   - 重要的卖压来源

### 分析工具

- CoinGecko/CoinMarketCap：基础供应数据
- Token Unlocks：解锁时间表
- 项目官方文档：详细代币经济学

## 年化收益率 APY/APR

### 核心概念

**APR（Annual Percentage Rate，年化利率）**：不考虑复利的年化收益率

**APY（Annual Percentage Yield，年化收益率）**：考虑复利效应的年化收益率

### 计算公式

**APR计算：**
```
APR = 周期收益率 × 一年的周期数
例：日收益0.1%，APR = 0.1% × 365 = 36.5%
```

**APY计算：**
```
APY = (1 + 周期收益率)^周期数 - 1
例：日收益0.1%，APY = (1 + 0.001)^365 - 1 ≈ 44%
```

### APR与APY的关系

```
APY = (1 + APR/n)^n - 1
```

其中n为一年的复利次数

**对比示例（假设APR=36.5%）：**

| 复利频率 | APY |
|----------|-----|
| 不复利 | 36.5% |
| 月复利 | 43.1% |
| 日复利 | 44.0% |
| 连续复利 | 44.1% |

### 在DeFi中的应用

1. **借贷协议**
   - 存款APY：存入资产获得的收益
   - 借款APR：借出资产需支付的利息
   - 利差是协议收入来源

2. **流动性挖矿**
   - 基础APR：交易费分成
   - 激励APR：代币奖励
   - 总APY：复利后的综合收益

3. **质押收益**
   - PoS网络的质押奖励
   - 通常以APR表示
   - 需考虑通胀影响

### 高收益的风险

**警惕超高APY的项目：**

1. **不可持续**
   - 早期高激励吸引资金
   - 收益会随时间下降
   - 代币价格下跌抵消收益

2. **无常损失**
   - 流动性挖矿特有风险
   - 价格波动导致损失
   - 高APY可能无法覆盖

3. **合约风险**
   - 高收益可能是骗局
   - 未经审计的合约
   - Rug Pull风险

### 计算实际收益

**真实收益考虑因素：**
- Gas费用
- 代币价格变化
- 无常损失
- 复投频率

```
实际收益 = 名义收益 - Gas费用 - 无常损失 ± 代币价格变化
```

## 夏普比率 (Sharpe Ratio)

### 核心概念

夏普比率由威廉·夏普提出，衡量投资组合每承受一单位风险所获得的超额收益，是评估风险调整后收益的重要指标。

### 计算公式

```
夏普比率 = (投资收益率 - 无风险利率) / 收益率标准差
```

**组成部分：**
- 投资收益率：实际获得的收益
- 无风险利率：通常用国债收益率
- 标准差：收益的波动性（风险）

### 解读方法

| 夏普比率 | 评价 |
|----------|------|
| < 0 | 收益低于无风险利率 |
| 0-1 | 一般 |
| 1-2 | 良好 |
| 2-3 | 优秀 |
| > 3 | 杰出（可能有问题） |

### 实际应用

1. **策略评估**
   - 比较不同策略的风险调整收益
   - 高夏普比率策略更优
   - 同样收益，波动小更好

2. **基金选择**
   - 评估基金经理能力
   - 超高夏普比率需警惕
   - 可能是短期运气或造假

3. **组合优化**
   - 最大化组合夏普比率
   - 资产配置的理论基础
   - 现代投资组合理论

### 计算示例

**策略A：**
- 年收益：20%
- 无风险利率：3%
- 标准差：15%
- 夏普比率 = (20% - 3%) / 15% = 1.13

**策略B：**
- 年收益：15%
- 无风险利率：3%
- 标准差：8%
- 夏普比率 = (15% - 3%) / 8% = 1.5

**结论**：策略B虽然收益低，但夏普比率更高，风险调整后表现更好

### 局限性

1. **假设正态分布**
   - 实际收益分布可能偏斜
   - 尾部风险被低估
   - 加密市场尤其不适用

2. **标准差的问题**
   - 上涨和下跌波动同等对待
   - 投资者通常只厌恶下跌波动
   - 可用索提诺比率替代

3. **历史数据**
   - 基于过去不代表未来
   - 策略失效时夏普比率无效
   - 需要足够长的数据周期

### 相关指标

1. **索提诺比率（Sortino Ratio）**
   - 只考虑下行波动
   - 更符合投资者风险偏好
   - 公式：(收益-无风险利率)/下行标准差

2. **卡尔玛比率（Calmar Ratio）**
   - 使用最大回撤作为风险
   - 适合评估绝对收益策略
   - 公式：年化收益/最大回撤

3. **信息比率（Information Ratio）**
   - 相对于基准的超额收益
   - 评估主动管理能力
   - 公式：超额收益/跟踪误差

## 总结

财务指标是投资分析的重要工具，但需要正确使用：

1. **综合分析**
   - 单一指标有局限性
   - 多指标交叉验证
   - 定量与定性结合

2. **行业对比**
   - 不同行业估值标准不同
   - 同行业横向比较更有意义
   - 了解行业特点

3. **动态视角**
   - 指标会随时间变化
   - 关注趋势而非单点数值
   - 预期比现状更重要

4. **深入研究**
   - 指标背后的原因
   - 会计政策的影响
   - 可持续性分析

5. **风险意识**
   - 任何指标都可能失灵
   - 保持独立思考
   - 分散投资降低风险

投资决策不能仅依赖指标，还需要对项目、团队、市场环境有全面了解。
