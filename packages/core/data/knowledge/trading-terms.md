# 交易术语

本文档介绍金融交易中常用的技术分析术语和交易概念，帮助用户理解市场分析方法和交易操作。

---

## K线图 (Candlestick Chart)

### 核心概念

K线图是一种用于展示金融资产价格变动的图表类型，起源于日本江户时代的米市交易。每根K线包含四个价格信息：开盘价、收盘价、最高价、最低价。

### K线组成

```
     │  ← 上影线（最高价）
     │
   ┌─┴─┐
   │   │ ← 实体（开盘价和收盘价之间）
   │   │
   └─┬─┘
     │
     │  ← 下影线（最低价）
```

**阳线（上涨）**：收盘价 > 开盘价，通常显示为绿色或红色
**阴线（下跌）**：收盘价 < 开盘价，通常显示为红色或绿色

注：中国市场习惯用红涨绿跌，国际市场多为绿涨红跌

### 常见K线形态

1. **锤子线/倒锤子线**
   - 长下影线，小实体
   - 可能预示反转
   - 出现在下跌趋势底部为看涨信号

2. **十字星**
   - 开盘价≈收盘价
   - 表示多空力量平衡
   - 可能预示趋势反转

3. **吞没形态**
   - 当前K线完全包含前一K线
   - 看涨吞没：阳线吞没阴线
   - 看跌吞没：阴线吞没阳线

4. **早晨之星/黄昏之星**
   - 三根K线组合
   - 重要的反转信号
   - 早晨之星看涨，黄昏之星看跌

### 时间周期选择

| 周期 | 适用场景 | 特点 |
|------|----------|------|
| 1分钟/5分钟 | 超短线交易 | 噪音大，信号多 |
| 15分钟/1小时 | 日内交易 | 平衡精度和趋势 |
| 4小时/日线 | 波段交易 | 趋势清晰 |
| 周线/月线 | 长期投资 | 过滤短期波动 |

### 使用注意事项

- 单根K线信号可靠性有限，需结合多根K线判断
- 关注关键位置（支撑/阻力位）的K线形态
- 结合成交量分析更加准确
- 不同市场环境下同一形态可能有不同含义

## MACD 指标

### 核心概念

MACD（Moving Average Convergence Divergence，指数平滑异同移动平均线）是一种趋势跟踪动量指标，通过计算两条移动平均线的差异来判断买卖时机。

### 计算方法

```
DIF = EMA(12) - EMA(26)    # 快线
DEA = EMA(DIF, 9)          # 慢线（信号线）
MACD柱 = 2 × (DIF - DEA)   # 柱状图
```

其中EMA为指数移动平均

### 组成部分

1. **DIF线（快线）**
   - 短期EMA与长期EMA的差值
   - 对价格变化反应灵敏

2. **DEA线（慢线/信号线）**
   - DIF的移动平均
   - 对价格变化反应较慢

3. **MACD柱状图**
   - DIF与DEA的差值
   - 直观显示两线距离

### 交易信号

1. **金叉（买入信号）**
   - DIF上穿DEA
   - 柱状图由负转正
   - 零轴上方金叉信号更强

2. **死叉（卖出信号）**
   - DIF下穿DEA
   - 柱状图由正转负
   - 零轴下方死叉信号更强

3. **顶背离（卖出预警）**
   - 价格创新高，MACD未创新高
   - 上涨动能减弱
   - 可能即将下跌

4. **底背离（买入预警）**
   - 价格创新低，MACD未创新低
   - 下跌动能减弱
   - 可能即将上涨

### 使用技巧

- 在趋势市场中使用效果较好
- 震荡市场中可能产生频繁假信号
- 结合其他指标确认信号
- 关注零轴位置判断大趋势

## RSI 指标

### 核心概念

RSI（Relative Strength Index，相对强弱指数）是一种动量振荡指标，用于衡量价格变动的速度和幅度，判断市场超买或超卖状态。

### 计算方法

```
RS = 平均上涨幅度 / 平均下跌幅度
RSI = 100 - 100/(1 + RS)
```

通常使用14周期计算

### 取值范围

RSI值在0到100之间波动：
- **RSI > 70**：超买区域，可能面临回调
- **RSI < 30**：超卖区域，可能面临反弹
- **RSI = 50**：多空平衡位置

### 交易信号

1. **超买超卖**
   - RSI进入超买区不一定立即下跌
   - RSI进入超卖区不一定立即上涨
   - 需要观察RSI走出极值区域确认

2. **背离信号**
   - 顶背离：价格新高，RSI未新高
   - 底背离：价格新低，RSI未新低
   - 背离是较强的反转信号

3. **中线交叉**
   - RSI上穿50：转为多头
   - RSI下穿50：转为空头

4. **失败摆动**
   - 在极值区域的反转形态
   - 不需要价格确认的独立信号

### 参数调整

| 周期 | 特点 | 适用场景 |
|------|------|----------|
| 7 | 灵敏度高，信号多 | 短线交易 |
| 14 | 标准设置，平衡 | 通用 |
| 21 | 平滑，信号少 | 中长线 |

### 使用注意事项

- 强势趋势中RSI可能长期处于超买/超卖区
- 不要仅凭超买超卖做逆势交易
- 结合趋势方向使用更有效
- 在震荡行情中效果较好

## 布林带 (Bollinger Bands)

### 核心概念

布林带是由约翰·布林格（John Bollinger）发明的技术分析工具，由三条线组成：中轨（移动平均线）和上下轨（标准差通道），用于衡量价格波动性和潜在的超买超卖状态。

### 计算方法

```
中轨 = SMA(20)                    # 20周期简单移动平均
上轨 = 中轨 + 2 × 标准差(20)
下轨 = 中轨 - 2 × 标准差(20)
```

### 核心特性

1. **自适应性**
   - 波动大时带宽扩张
   - 波动小时带宽收窄
   - 自动适应市场波动

2. **统计意义**
   - 约95%的价格落在带内
   - 价格触及边界为统计异常
   - 但不保证立即回归

### 交易策略

1. **区间震荡策略**
   - 价格触及上轨考虑卖出
   - 价格触及下轨考虑买入
   - 适用于横盘震荡市场

2. **趋势突破策略**
   - 价格突破上轨可能开启上涨趋势
   - 价格突破下轨可能开启下跌趋势
   - 需要成交量确认

3. **带宽收窄策略**
   - 带宽极度收窄预示大行情
   - 方向需要等待突破确认
   - 收窄后通常有剧烈波动

4. **走带策略**
   - 强趋势中价格沿上轨或下轨运行
   - 不要过早逆势
   - 等待明确的反转信号

### %B指标

```
%B = (价格 - 下轨) / (上轨 - 下轨)
```

- %B > 1：价格在上轨之上
- %B = 0.5：价格在中轨
- %B < 0：价格在下轨之下

### 使用注意事项

- 触及边界不等于必须反转
- 结合趋势方向判断
- 关注带宽变化预测波动
- 可以与其他指标配合使用

## 支撑位与阻力位

### 核心概念

支撑位和阻力位是技术分析的基础概念：
- **支撑位**：价格下跌时可能遇到买盘支撑的价位
- **阻力位**：价格上涨时可能遇到卖盘阻力的价位

### 形成原因

1. **心理因素**
   - 整数关口（如10000、50000）
   - 投资者容易在这些位置设置订单

2. **历史交易**
   - 前期高点成为阻力
   - 前期低点成为支撑
   - 交易密集区形成支撑/阻力

3. **技术指标**
   - 移动平均线
   - 斐波那契回撤位
   - 趋势线

### 识别方法

1. **水平支撑阻力**
   - 多次测试的价位
   - 测试次数越多越有效
   - 突破需要成交量确认

2. **趋势线支撑阻力**
   - 连接低点形成上升趋势线（支撑）
   - 连接高点形成下降趋势线（阻力）
   - 需要至少两点确认

3. **移动平均线**
   - MA20/MA50/MA200常作为动态支撑阻力
   - 价格回踩均线后反弹
   - 均线在不同周期有不同意义

4. **斐波那契回撤**
   - 23.6%、38.2%、50%、61.8%、78.6%
   - 常见的回撤支撑位
   - 黄金分割比例

### 交易应用

1. **支撑位买入**
   - 价格回落到支撑位时寻找买入机会
   - 设置止损在支撑位下方
   - 支撑不破可继续看涨

2. **阻力位卖出**
   - 价格上涨到阻力位时考虑减仓
   - 阻力不破可能回落
   - 突破后可能加速上涨

3. **支撑阻力转换**
   - 支撑位被跌破后变成阻力位
   - 阻力位被突破后变成支撑位
   - 重要的交易规则

### 注意事项

- 支撑阻力是区域而非精确价格
- 需要留出一定容差空间
- 假突破较为常见
- 结合成交量和其他指标判断

## 交易量 (Volume)

### 核心概念

交易量是指在特定时间段内交易的资产数量，是验证价格走势的重要指标。成交量反映市场参与者的活跃程度和交易意愿。

### 量价关系基础

1. **量增价涨**
   - 健康的上涨趋势
   - 买盘积极，趋势可能延续
   - 牛市典型特征

2. **量增价跌**
   - 恐慌性抛售
   - 下跌趋势确认
   - 可能加速下跌

3. **量缩价涨**
   - 上涨乏力
   - 可能接近顶部
   - 谨慎追高

4. **量缩价跌**
   - 下跌动能减弱
   - 可能接近底部
   - 关注反转信号

### 成交量指标

1. **OBV（能量潮）**
   - 累积成交量指标
   - 价格上涨加成交量，下跌减成交量
   - 判断资金流向

2. **成交量移动平均**
   - 对比当前成交量与平均值
   - 判断成交量是否异常
   - 识别放量和缩量

3. **换手率**
   - 成交量 / 流通量
   - 衡量市场活跃度
   - 高换手率可能预示变盘

### 成交量分析技巧

1. **突破确认**
   - 有效突破应伴随放量
   - 无量突破可信度低
   - 假突破往往成交量小

2. **背离分析**
   - 价格创新高，成交量萎缩
   - 可能是上涨末期信号
   - 同样适用于下跌趋势

3. **底部特征**
   - 下跌末期成交量萎缩
   - 反转时伴随放量
   - "地量见地价"

4. **顶部特征**
   - 上涨末期可能出现天量
   - 高位放量滞涨危险
   - "天量见天价"

### 不同市场的成交量

- **加密货币市场**：24小时交易，关注近期平均量
- **股票市场**：开盘和收盘时段成交活跃
- **外汇市场**：去中心化，真实成交量难以统计

## 做多与做空

### 核心概念

**做多（Long）**：买入资产，预期价格上涨后卖出获利
**做空（Short）**：借入资产卖出，预期价格下跌后买回获利

### 做多详解

操作流程：
1. 以当前价格买入资产
2. 持有等待价格上涨
3. 以更高价格卖出
4. 差价即为利润

风险特点：
- 最大亏损：投入本金的100%
- 理论最大收益：无限

### 做空详解

操作流程：
1. 从交易所或他人借入资产
2. 以当前价格卖出
3. 价格下跌后以更低价格买回
4. 归还借入的资产
5. 差价扣除利息即为利润

风险特点：
- 理论最大亏损：无限（价格可以无限上涨）
- 最大收益：做空价格（价格最多跌到零）

### 杠杆交易

**杠杆倍数**：放大仓位的倍数
- 2x杠杆：用100美元控制200美元仓位
- 10x杠杆：用100美元控制1000美元仓位

**强制平仓（爆仓）**
- 亏损达到保证金时强制平仓
- 杠杆越高，容错空间越小
- 计算：强平价格 = 开仓价 × (1 ± 1/杠杆倍数)

### 合约类型

1. **永续合约**
   - 没有到期日
   - 通过资金费率锚定现货价格
   - 加密货币市场最常用

2. **交割合约**
   - 有固定到期日
   - 到期时按结算价交割
   - 传统期货市场常用

### 风险管理

1. **仓位控制**
   - 单笔交易不超过总资金的2-5%
   - 根据胜率和盈亏比调整

2. **杠杆选择**
   - 新手建议不超过3x
   - 高杠杆适合短线小仓位
   - 永远为波动留出空间

3. **止损设置**
   - 开仓前确定止损位
   - 严格执行，不随意移动
   - 止损是保护本金的关键

## 止损与止盈

### 核心概念

**止损（Stop Loss）**：预设亏损上限，价格触及自动平仓，控制损失
**止盈（Take Profit）**：预设盈利目标，价格触及自动平仓，锁定利润

### 止损的重要性

1. **保护本金**
   - 单笔亏损有限
   - 避免深度套牢
   - 保持交易能力

2. **执行纪律**
   - 消除情绪干扰
   - 避免侥幸心理
   - 系统化交易

3. **风险可控**
   - 计算最大风险敞口
   - 合理分配仓位
   - 长期生存的基础

### 止损设置方法

1. **固定百分比止损**
   - 例如：亏损2%止损
   - 简单易执行
   - 适合新手

2. **技术位止损**
   - 设在支撑位下方
   - 设在趋势线下方
   - 更合理但需要分析

3. **ATR止损**
   - 基于平均真实波幅
   - 自适应市场波动
   - 公式：止损 = 入场价 - N × ATR

4. **时间止损**
   - 持仓一定时间未盈利则平仓
   - 避免资金被长期占用
   - 适合短线策略

### 止盈策略

1. **固定目标止盈**
   - 设定固定盈利目标
   - 例如：盈利5%止盈
   - 简单明确

2. **盈亏比止盈**
   - 止盈距离 = N × 止损距离
   - 常用2:1或3:1
   - 确保长期正期望

3. **移动止盈**
   - 随价格上涨移动止盈位
   - 保护浮盈
   - 让利润奔跑

4. **分批止盈**
   - 分多次部分平仓
   - 例如：涨10%平50%，涨20%平剩余
   - 平衡锁定利润与追求更大收益

### 常见错误

1. **不设止损**
   - 抱有侥幸心理
   - 小亏变大亏
   - 最终爆仓或深套

2. **止损过紧**
   - 频繁被震出局
   - 错过行情
   - 增加交易成本

3. **移动止损**
   - 亏损时放宽止损
   - 违背初衷
   - 纪律性差

4. **不止盈**
   - 贪婪导致利润回吐
   - 浮盈变浮亏
   - 应该锁定合理利润

### 盈亏比与胜率

| 胜率 | 需要盈亏比 | 示例 |
|------|------------|------|
| 30% | > 2.33:1 | 高盈亏比策略 |
| 40% | > 1.5:1 | 趋势跟踪策略 |
| 50% | > 1:1 | 平衡策略 |
| 60% | > 0.67:1 | 高胜率策略 |

公式：期望值 = 胜率 × 平均盈利 - 败率 × 平均亏损

## 总结

技术分析是辅助决策的工具，不是预测未来的水晶球：

1. **综合分析**：不要依赖单一指标，多指标综合判断
2. **顺势而为**：识别趋势，顺势交易成功率更高
3. **风险第一**：先考虑可能亏多少，再考虑能赚多少
4. **纪律执行**：设定规则并严格执行
5. **持续学习**：市场在变化，方法也需要优化

技术分析需要大量实践才能熟练运用，建议先用模拟盘练习。
